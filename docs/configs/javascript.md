## Options

### strict

Enable strict checking for JavaScript files.

- **Type**: `boolean`
- **Default**: `false`

### ecmaVersion

The ECMAScript version of the code being linted.

- **Type**: `Linter.EcmaVersion`
- **Default**: `'latest'`

- **Type**: `Rules`
