# Sort

## Options

### additionalJsonFiles

Additional JSON files to sort all properties alphabetically.

- **Type**: `string[]`

### additionalYamlFiles

Additional YAML files to sort all properties alphabetically.

- **Type**: `string[]`

### i18nLocale

Enable i18n locale sorting.

- **Type**: `boolean` (default: true)
