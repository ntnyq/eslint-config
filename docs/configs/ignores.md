# Ignores

## Built-in ignore globs

```shell

# ==================== Dependencies and Build Output ====================
# Dependencies
**/node_modules/**
**/.pnpm-store/**

# Build output
**/dist/**
**/output

# ==================== Lock Files ====================
# Package manager lock files
**/package-lock.json
**/yarn.lock
**/pnpm-lock.yaml
**/bun.lock?(b)
**/deno.lock

# ==================== Force Lint Directories ====================
# These directories are excluded from ignore using !
!.github
!.vitepress
!.vuepress
!.vscode

# ==================== Documentation and License Files ====================
# Changelogs
**/CHANGELOG*.md

# License files
**/LICENSE*

# ==================== Minified Files ====================
# Minified files
**/*.min.*

# ==================== Tests and Snapshots ====================
# Test snapshots
**/__snapshots__

# Test coverage
**/coverage

# Test fixtures
**/fixtures

# ==================== Auto-generated Type Files ====================
# Generated by unplugin-auto-import
**/auto-import?(s).d.ts

# Generated by unplugin-vue-components
**/components.d.ts

# Generated by unplugin-vue-router
**/typed-router.d.ts

# Generated by vite-plugin-uni-pages
**/uni-pages.d.ts

# ==================== Static Assets ====================
# Public static assets
**/public
**/static

# ==================== Temporary Files and Cache ====================
# VitePress and VuePress temp files
**/?(.)temp
**/?(.)cache

# ESLint cache
**/.eslintcache

# Stylelint cache
**/.stylelintcache

# ==================== Framework and Tool Generated Directories ====================
# ESLint config inspector
**/.eslint-config-inspector

# Vite inspect tool
**/.vite-inspect

# Nuxt
**/.nuxt

# WXT
**/.wxt

# Output directory
**/.output

# tsup build tool
**/.tsup

# Nitro
**/.nitro

# Vercel
**/.vercel

# Cloudflare Wrangler
**/.wrangler

# Changesets
**/.changeset

# ==================== Configuration Files ====================
# npm/yarn config
**/.npmrc
**/.yarnrc

# Git hooks
**/.husky

# ==================== Tauri Specific ====================
# Generated by Tauri
**/src-tauri/gen
**/src-tauri/target
```

Check [source file](https://github.com/ntnyq/eslint-config/blob/main/src/globs.ts) for detail.

## Options

Passing an array to extend the built-in ignores, or passing a function to modify the built-in ignores.

### customIgnores

Custom ignore patterns.

- **Type**: `string[] | ((ignores: string[]) => string[])`
- **Default**: `[]`
